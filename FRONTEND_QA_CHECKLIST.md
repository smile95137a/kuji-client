## 快速測試 SOP

```
1. 清除 localStorage → 以未登入狀態執行 P 組（路由守衛）
2. 註冊新帳號 → 走 A-11 ~ A-23（含 Email 驗證完整流程）
3. 登入後逐一測試 D 組（會員中心所有子頁）
4. 在賞品盒執行 D-25 ~ D-30（出貨完整流程，含配送方式選擇）
5. 測試 B 組公開頁面，確認 R-05（themes 改為 GET）
6. 執行 npm run build 確認 X-01、X-02
```
